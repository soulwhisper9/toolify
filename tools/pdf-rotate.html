Frequently asked questions

Can I rotate only specific pages?
Yes — click pages to select them, then use the "Selected" rotation buttons to rotate only those pages.
Is the original file changed?
No. You download a new rotated PDF. Your original file is untouched.
Does rotation reduce quality?
No. PDF rotation is a metadata change — content is not re-rendered or recompressed.
