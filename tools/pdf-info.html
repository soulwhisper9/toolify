Frequently asked questions

Is my PDF file uploaded anywhere?
No. The file is read locally in your browser using pdf-lib. Nothing is sent to any server.
What page size formats does this detect?
It shows exact dimensions in points. A4 is 595×842pt, US Letter is 612×792pt.
Can it read password-protected PDFs?
Basic info like page count and file size is shown, but metadata may be restricted for encrypted PDFs.
