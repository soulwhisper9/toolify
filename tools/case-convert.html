Frequently asked questions

What is camelCase used for?
camelCase is used in programming for variable names (e.g. firstName, totalAmount). The first word is lowercase and each subsequent word starts with uppercase.
What is the difference between Title Case and Sentence case?
Title Case capitalizes the first letter of every word. Sentence case only capitalizes the first letter of the first word, like a normal sentence.
What is snake_case used for?
snake_case uses underscores between words with all lowercase. It's common in Python, database column names, and file naming conventions.
